# Dev-2 Arrow HTML Ödevi

## Proje Hakkında

Bu proje, Web Programlama dersi kapsamında hazırlanmış basit bir HTML web sitesi çalışmasıdır.
Projede temel HTML yapıları kullanılarak birden fazla sayfadan oluşan bir web sitesi oluşturulmuştur.

## Proje Yapısı

Proje aşağıdaki dosya ve klasörlerden oluşmaktadır:

```
dev-2-arrow-cahit/
│
├── index.html
├── hakkimda.html
└── yazilar/
    └── yazi-1.html
```

### index.html

Web sitesinin ana sayfasıdır.

### hakkimda.html

Site sahibi ile ilgili bilgilerin bulunduğu sayfadır.

### yazilar/yazi-1.html

Yazılar klasörü içinde bulunan örnek bir yazı sayfasıdır.

## Amaç

Bu projenin amacı:

* BAsit HTML sayfa yapısını öğrenmek
* Çok sayfalı basit bir web sitesi oluşturmak
* Git ve GitHub kullanarak versiyon kontrolü pratiği yapmaktır.
